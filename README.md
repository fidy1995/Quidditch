# Quidditch
The Quidditch game made for SE344-Computer Graphics.

## Controllers:
Up - Decrease force
Down - Increase force
Left - Spin bat left
Right - Spin bat right

W - View up
S - View down
A - View left
D - View right

U - Move forward
J - Move backward
H - Move left
K - Move right
Page Up - Move closer
Page Down - Move farther

R - Reset Views
L - Turn on/off light
F11 - Full Screen / Windowed
Esc - Quit game
